描述常数操作的指标
1.冒泡排序 长度n 共进行(a1+an)*n/2次交换操作
(a1+an)*n/2 = (2*a1+(n-1)*d)*n/2 = a1*n + (n^2-n)*d/2 = d/2*(n^2) + ((2*a1-d)/2)*n
a1 d均为常数 取变量n的最高阶 即为时间复杂度O(n^2)

2.选择排序 长度n 共进行(a1+an)*n/2次取数比较操作
同1 时间复杂度O(n^2)

3.二分查找 长度n 共进行
时间复杂度O(log2 n)
